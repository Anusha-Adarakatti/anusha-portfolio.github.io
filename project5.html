Music Generation with Deep Learning

This research project implements advanced neural architectures for algorithmic music generation, exploring both classical and jazz styles through LSTM networks and Variational Autoencoders (VAE). The system demonstrates how deep learning can capture musical patterns, styles, and structures to generate original compositions.

This innovative project explores two distinct approaches to AI-powered music generation:

  • üéπ LSTM Neural Networks for classical piano music generation in the style of Chopin
  • üé∑ Convolutional Variational Autoencoder (CVAE) for jazz music generation
  • üß† Dual Architecture Design: Each model specializes in different musical characteristics
  • üéº End-to-End Pipeline: From raw MIDI/audio data to playable generated compositions

LSTM Model Architecture

  • üî¢ Input Processing: MIDI files parsed into note sequences using music21
  • üß† Network Structure:
    • First LSTM layer: 512 units with return sequences
    • Dropout layer (0.1) for regularization
    • Second LSTM layer: 256 units
    • Dense layer: 256 units with ReLU activation
    • Output layer: Dense with softmax activation
  • ‚ö° Training: 200 epochs with Adamax optimizer

VAE Model Architecture

  • üéöÔ∏è Input Processing: Audio converted to spectrograms using librosa
  • üåê Network Structure:
    • Encoder: 1D convolutional layers with ResNet-inspired blocks
    • Latent Space: 2D for visualization and sampling

    • Custom ResNet1DBlocks for feature extraction
  • ‚ö° Training: 20 epochs using Adam optimizer

üìä Datasets & Preprocessing

  • üéº Classical Dataset:
    • Chopin's MIDI files from classical-music-midi/chopin/
    • Parsed into note sequences with timing information
    • Sequences of 40 notes used as input features
  • üé∑ Jazz Dataset:
    • GTZAN Dataset for Music Genre Classification
    • Specifically jazz genre samples
    • Processed into spectrograms for VAE input
  • üßπ Data Cleaning: Normalized note values, standardized audio sample rates

üéπ Music Generation Process

  • üéº LSTM Generation:
    • Starts with a seed sequence from training data
    • Predicts one note at a time using previous outputs
    • Converts resulting sequence to MIDI format
    • Can generate arbitrary length compositions
  • üé∑ VAE Generation:
    • Samples from learned latent space distribution
    • Decodes samples into audio spectrograms
    • Converts spectrograms to playable audio format
    • Enables interpolation between musical styles

  • üìä Visualization: Includes sheet music (LSTM) and waveform plots (VAE)
